TokyoWesterns CTF 4th 2018

About

TokyoWesterns CTF is a security competition hosted by TokyoWesterns. The winner would be qualified to CODE BLUE CTF 2018 Finals!

This contest is sponsored by Google Cloud and it uses the Google Cloud Platform.

Rules/Prohibitions

  • Team competition
  • Team capacity is unlimited.
  • The flag format is "TWCTF{something}" unless otherwise specified.
  • You will get points after submitting a flag to the score server.
  • Some challenges may have multiple flags.
  • Earlier team would get higher rank if some teams had same points.
  • If you have any questions during the content, please contact us on IRC. (refer to "IRC Information" to get more details.)

Prohibitions

  • Do not attack/obstruct other teams.
  • Do not attack any other servers except *.chal.ctf.westerns.tokyo.
  • Do not share answers/hints with other teams.
  • Be Kind to server resources.
